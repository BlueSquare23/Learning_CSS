Displaying an Image

Use the "img" tag to link to an image in an html document.
For Example,

<img src="URL" alt="image description" >

The "img" tag takes two attributes, "src" & "alt"

* "src" stands for source and this is the location of the image.

* "alt" stands for Alternate Text and it is often used as a simple description of the image. This can be useful for people who use sceen readers or if for some reason the image is not able to be rendered.

Displaying a List

There are two types of lists in HTML. Ordered lists use the "ol" tag and are displayed as a numbered set of items. Unordered lists use the "ul" tag and are displayed as a list of bullet points. Both kinds of lists take the "li" attribute which stands for list item.

* Source for ordered lists 

         <ol>
            <li>List Item</li>
         </ol>

Displaying a Table

The basic tage for building tables is the "table" tag. To create rows and columns within the table use the "tr" (table row) tag followed by the "td" (table data) tag. To give your table a border use the 'border="1"' attribute within the <table>.
For Example,

* Source

         <table border="1">
            <tr>
               <td>Content</td>
            </tr>
         </table>

* Rendered

Content Content Content

You can add additional rows to your tables by stacking the <tr> tags. You can also use the "th" (table headers) tag for columns which you'd like to be headers.
